Lemma 4.4
#

If \(\sum _{k = 0}^{\infty }a_k\) is convergent, then

\[ \lim _{k \to \infty } a_k = 0. \]
Proof ▶

Let \(S_n = \sum _{k = 0}^{n}a_k\), then \(\lim _{n \rightarrow \infty }S_n = s\) or some \(s \in \mathbb {R}\). Write \(a_n = S_n - S_{n - 1}\) for \(n \geq 1\). By COLT \(\lim _{n \to \infty }a_n = s - s = 0\).

Lemma 4.5 Harmonic Series
#

The series

\[ \sum _{k = 1}^{\infty }\frac{1}{n} \]

diverges.

Proof ▶

Let \(S_n = \sum _{k = 1}^{n}\frac{1}{k}\) then

\[ S_{2 ^n} \geq 1 + \frac{n}{2} \]

so the sequence of partial sums diverges. Thus the series diverges.

2 Convergence Criteria

Theorem 4.8 Comparison Test
#

Let \(N \in \mathbb {N}\) let \((a_k)_{k \in \mathbb {N}_0}\), \((b_k)_{k \in \mathbb {N}_0}\) sequences with

\[ 0 \leq a_k \leq b_k \text{ for } k \geq N. \]

  1. If \(\sum _{k = 0}^{\infty }b_k\) is convergent, then \(\sum _{k = 0}^{\infty }a_k\) is convergent.

  2. If \(\sum _{k = 0}^{\infty }a_k\) is divergent, then \(\sum _{k = 0}^{\infty }b_k\) is divergent.

Proof ▶

Use \(\sum _{k = 0}^{\infty }a_k\) is convergent if and only if \(\sum _{k = N}^{\infty }a_k\) is convergent.

For \(n \geq N\) let \(s_n = \sum _{k = N}^{n}a_k\) and \(t_n = \sum _{k = N}^{n}b_k\). Theses are monotonically increasing.

If \(\sum _{k = 0}^{\infty }b_k\) is convergent so is \((t_n)_{n \geq N}\) to \(t\).

\[ s_n \leq t_n \leq t. \]

The sequence \(s_n\) is monotonically increasing by \(t\), so is convergent by Thm 2.16.

(b) Since \(s_n\) is monotonically increasing, divergence means \((s_n)\) is unbounded, but then \((t_n)\) is also unbounded.

Definition 4.10
#

A series \(\sum _{k = 1}^{\infty }a_k\) is called alternating, if

\[ a_{2k} \geq 0\text{ and } a_{2k - 1} \leq 0\text{ for all $k \in \mathbb {N}$} \]

or if

\[ a_{2k} \leq 0\text{ and } a_{2k - 1} \geq 0\text{ for all $k \in \mathbb {N}$}. \]
Theorem 4.11 Alternating Sign Test
#

Let \((a_k)_{k \in \mathbb {N}}\) be a monotonically decreasing sequence of positive numbers with \(\lim _{k \to \infty }a_k = 0\). Then the alternating series \(\sum _{k = 1}^{\infty }(-1) ^{k + 1}a_k\) is convergent.

For the sequence of partial sums \((s_n)_{n \in \mathbb {N}}\) we have

\[ S_2 \leq S_4 \leq \dotsi \leq S_{2n} \leq \dotsi \leq \sum _{k = 1}^{\infty }(-1) ^{k + 1}a_k \leq \dotsi \leq S_{2n - 1} \leq \dotsi \leq S_3 \leq S_1 \]

and

\[ \left|S_n - \sum _{k = 1}^{\infty }(-1) ^{k + 1}a_k\right| \leq a_{n + 1}. \]
Proof ▶

3 Absolute Convergence

Definition 4.12 Absolute convergence
#

Let \(\displaystyle \sum _{k = 1}^{\infty }a_k\) be a series, we call it absolutely convergent, if \(\displaystyle \sum _{k = 1}^{\infty }|a_k|\) is convergent.

Theorem 4.13 Absolute convergence implies convergence
#

Let \(\sum _{k = 1}^{\infty }a_k\) be absolutely convergent. Then the series is convergent.

Proof ▶

We have \(\sum _{k = 1}^{\infty }|a_k|\) is convergent. By COLT

\[ \sum _{k = 1}^{\infty }2|a_k|\text{ is convergent}. \]

We have \(-|a_k| \leq a_k \leq |a_k|\). So \(0 \leq |a_k| + a_k \leq 2|a_k|\). By the comparison test,

\[ \sum _{k = 1}^{\infty }(a_k + |a_k|) \]

is convergent. By COLT

\[ \sum _{k = 1}^{\infty }a_k = \sum _{k = 1}^{\infty }((a_k + |a_k|) - |a_k|) \]

is convergent. Additionally,

\[ \sum _{k = 1}^{\infty }a_k \leq \sum _{k = 1}^{\infty }|a_k| \]

we have

\[ \sum _{k = 1}^{n}a_k \leq \sum _{k = 1}^{n}|a_k|. \]
Theorem 4.14 Ratio Test
#

Let \((a_k)_{k \in \mathbb {N}}\) be a sequence with \(a_k \neq 0\) for all \(k \in \mathbb {N}\) except finitely many.

  1. If \(\displaystyle \lim _{k \to \infty }\left|\frac{a_{k + 1}}{a_k}\right| {\lt} 1\), then the series \(\displaystyle \sum _{k = 1}^{\infty }\) is absolutely convergent.

  2. If \(\displaystyle \lim _{k \to \infty }\left|\frac{a_{k + 1}}{a_k}\right| {\gt} 1\), then the series \(\displaystyle \sum _{k = 1}^{\infty }a_k\) is divergent.

Theorem 4.15 Root test
#

For a sequence \((a_k)_{k \in \mathbb {N}}\) set

\[ a = \limsup _{k \to \infty }\sqrt[k]{|a_k|} \]

  1. If \(a {\lt} 1\), then \(\displaystyle \sum _{k = 1}^{\infty }a_k\) is absolutely convergent.

  2. If \(a {\gt} 1\), then \(\displaystyle \sum _{k = 1}^{\infty }a_k\) is divergent.

Proof ▶

Assume \(a {\lt} 1\). Then for all but finitely many \(k\) we have

\[ \sqrt[k]{|a_k|} \leq q {\lt} 1 \]

use \(q = \frac{a + 1}{2}\). So for all \(k \geq n_0\) we have \(|a_k| \leq q ^k\). By comparison test with the convergent geometric series

\[ \sum _{k = 1}^{\infty }q ^k \]

we get absolute convergence.

Assume \(a {\gt} 1\), then for all but finitely many \(k\), \(\sqrt[k]{|a_k|} \geq q {\gt} 1\). hence for all \(k \geq n_1\) \(|a_k| \geq q ^k\). By the comparison test with the diverging geometric series

\[ \sum _{k = 1}^{\infty }q ^k \]

we get divergence.
