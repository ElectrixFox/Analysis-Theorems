Definition 1.1 Bounded above
✓
#

Lean declarations

  • bound_above

Given a set \(X\), \(X\) is bounded above if there exists a \(C\) such that for all \(x \in X\) \(x \leq C\).

Definition 1.4 Maximum
✓
#

Lean declarations

  • maximum

Let \(X \subset \mathbb {R}\) be bounded above. A number \(C \in \mathbb {R}\) is called a maximum of \(X\) if \(C\) is an upper bound of \(X\) and for all \(x \in X\), \(x \leq C\). In other words, \(C\) is a maximum if \(\forall x \in X, x \leq C\).

Definition 1.5 Minimum
✓
#

Definition 1.6 Supremum
✓
#

Lean declarations

  • supremum

Let \(X \subset \mathbb {R}\) be bounded above. A number \(C \in \mathbb {R}\) is called a supremum of \(X\) if \(C\) is an upper bound of \(X\) and whenever \(B\) is an upper bound of \(X\), then \(C \leq B\).
